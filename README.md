# Dotfiles

A simple repository to manage my configuration files.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/rakeshravilla/dotfile.git ~/dotfiles
    cd ~/dotfiles
    ```
2.  Run the installation script:
    ```bash
    ./install/install.sh
    ```

This will automatically detect your OS (macOS or WSL) and install the necessary packages and configurations.
